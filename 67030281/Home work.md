หมุน POT ให้ตรงกับมุมบนกระดาษ แล้วบันทึกค่า ADC

| มุม (°) | ค่า ADC(คาดการณ์) | ค่า ADC ที่อ่านได้ |
| :-----: | :---------------: | ------------------ |
|    0    |        180        |          183       |
|   45    |       1100        |         1096       |
|   90    |       2100        |         2164       |
|   135   |       3000        |         3097       |
|   180   |       3900        |         4095       |

จากข้อมูลนี้ให้หาค่า

- **$ADC_{min}$** = ค่า ADC ที่มุม 0°
- **$ADC_{max}$** = ค่า ADC ที่มุมสูงสุด
- **$θ_{max}$** = มุมสูงสุด เช่น 180°
