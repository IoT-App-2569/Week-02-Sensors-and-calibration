# เก็บข้อมูลคาลิเบรต

หมุน POT ให้ตรงกับมุมบนกระดาษ แล้วบันทึกค่า ADC

| มุม (°) | ค่า ADC(คาดการณ์) | ค่า ADC ที่อ่านได้ |
| :-----: | :---: | :---:|
|    0    | 180  | 0 |
|   45    | 1100 | 50 |
|   90    | 2100 | 830 |
|   135   | 3000 | 1707 |
|   180   | 3900 | 2300 |
