# ผลการทดลอง
###  **ขั้นตอนที่ 4 เก็บข้อมูลคาลิเบรต**

หมุน POT ให้ตรงกับมุมบนกระดาษ แล้วบันทึกค่า ADC

| มุม (°) | ค่า ADC(คาดการณ์) | ค่า ADC ที่อ่านได้ |
| :-----: | :---------------: | ------------------ |
|    0    |        180        |         190           |
|   45    |       1100        |         960          |
|   90    |       2100        |         1800          |
|   135   |       3000        |          2800          |
|   180   |       3900        |           3600         |
